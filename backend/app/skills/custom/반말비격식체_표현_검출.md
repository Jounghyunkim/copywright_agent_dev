# Prompt Template

다음의 카피라이팅 문구({{copies}})가 마케팅 브리프({{brief}}), 분석 리포트({{analysis_report}}), 전략 메시지({{strategic_message}})에 맞게 작성되었는지 평가해 주십시오. 주요 평가 기준은 다음과 같습니다:

1. 반말(비격식체) 또는 속어, 지나치게 친근하거나 공식적이지 않은 표현이 포함되어 있는지 검출합니다.
2. 모든 문구가 격식체(존댓말, 공식적 언어)로 작성되어야 하며, 반말이나 비격식 표현이 없어야 합니다.
3. 속어, 은어, 줄임말 등 마케팅 커뮤니케이션에서 부적절한 표현이 사용되지 않았는지 확인합니다.

점수 산정 기준(0-100점):
- 100점: 반말, 비격식, 속어, 은어, 줄임말이 전혀 없음. 모든 문구가 격식체임.
- 80점 이상: 미세한 비격식 표현이 있으나, 전체적으로 공식성을 해치지 않음.
- 60점 이상: 일부 비격식/속어가 있으나, 수정하면 공식 커뮤니케이션에 적합.
- 60점 미만: 다수의 반말, 속어, 은어, 줄임말이 포함되어 공식적 커뮤니케이션에 부적합.

합격 기준: 80점 이상일 경우 pass, 미만은 fail로 처리합니다.

좋은 예시(합격):
- "뒤로 이동"

나쁜 예시(불합격):
- "빠꾸"

평가 결과는 아래 형식으로 요약해 주세요.

Output Format (JSON):
{
  "score": 점수 (0-100),
  "passed": true/false,
  "strengths": [문구의 장점],
  "weaknesses": [문구의 단점],
  "improvements": [개선 방안]
}
