광고 카피의 창의성, 주목도, 의외성, 감정 유발 강도를 다차원으로 평가한다. 이 스킬은 hard_fail 없이 참고 지표로만 사용되며, 카피라이터와 크리에이티브 디렉터가 카피의 크리에이티브 임팩트를 객관적으로 비교 판단할 수 있도록 돕는다.

## 사용 조건

이 스킬은 다음 상황에서 트리거된다.

- 카피 초안이 완성된 후 크리에이티브 품질을 정량적으로 평가하고자 하는 경우
- 다수의 카피 후보 중 크리에이티브 임팩트가 가장 높은 안을 선별하고자 하는 경우
- A/B 테스트 전 카피 후보들의 예상 주목도를 사전 비교하는 경우
- 캠페인 사후 분석에서 크리에이티브 점수와 실제 성과의 상관관계를 분석하는 경우
- 카피라이터 역량 개발을 위한 피드백 자료로 활용하는 경우

## 입력

필수 입력 필드:

| 필드명 | 타입 | 설명 |
|--------|------|------|
| `copy_text` | `dict` | 평가 대상 카피. `headline`, `body`, `cta` 포함 |

선택 입력 필드:

| 필드명 | 타입 | 설명 |
|--------|------|------|
| `target_audience` | `str` | 타겟 오디언스. 오디언스에 따라 임팩트 기대치가 달라짐 |
| `category_benchmarks` | `dict` | 동일 카테고리/산업의 크리에이티브 벤치마크 점수 |
| `campaign_objective` | `str` | 캠페인 목표. "인지도 향상", "전환율 증대", "브랜드 선호도 강화" 등 |
| `comparison_copies` | `list[dict]` | 비교 대상 카피 목록. 상대적 평가에 활용 |
| `channel` | `str` | 게재 채널. 채널별 크리에이티브 기대치 차이 반영 |

## 출력

주요 출력:

| 필드명 | 타입 | 설명 |
|--------|------|------|
| `attention_score` | `float (0.0-1.0)` | 주목도 점수. 스크롤/피드 환경에서 시선을 끌 수 있는 정도. 강렬한 첫 문장, 시각적 언어, 숫자/기호 활용도 평가 |
| `surprise_score` | `float (0.0-1.0)` | 의외성 점수. 예상을 벗어나는 표현, 반전, 창의적 비유 등의 활용도 평가 |
| `emotion_score` | `float (0.0-1.0)` | 감정 유발 점수. 공감, 기쁨, 호기심, 안심 등 감정 반응을 일으키는 강도 평가 |
| `memorability_score` | `float (0.0-1.0)` | 기억 잔존도 점수. 라임, 리듬, 반복, 독특한 조어 등 기억에 남는 요소 평가 |
| `originality_score` | `float (0.0-1.0)` | 독창성 점수. 클리셰 회피도, 새로운 표현 시도, 차별화된 관점 활용도 평가 |
| `overall_impact` | `float (0.0-1.0)` | 5개 항목의 가중 평균 종합 임팩트 점수 |
| `score_rationale` | `dict` | 각 점수의 평가 근거. 항목별 강점과 약점 설명 |
| `creative_tips` | `list[str]` | 크리에이티브 임팩트 향상을 위한 구체적 팁 |

**참고**: 이 스킬은 hard_fail을 발생시키지 않는다. 모든 점수는 참고 지표로만 사용된다.

## 실행 단계

1. **주목도 평가** (`attention_score`):
   - 첫 3단어/5자의 임팩트 분석 (스크롤 환경에서의 후킹 효과)
   - 숫자, 기호, 대조적 표현 사용 여부
   - 시각적으로 눈에 띄는 문장 구조 (짧은 문장, 파격적 배치)
   - 질문형/도발적 표현 사용도
2. **의외성 평가** (`surprise_score`):
   - 상투적 표현(클리셰) 사용 빈도 (많을수록 감점)
   - 예상 밖의 단어 조합, 반전 구조
   - 비유, 은유의 신선함
   - 카테고리 관습을 벗어나는 표현
3. **감정 유발 평가** (`emotion_score`):
   - 감정 키워드 밀도와 강도
   - 공감 요소 (타겟 오디언스의 페인포인트 건드림)
   - 스토리텔링 요소 활용도
   - 긍정/부정/긴급성 등 감정 유형 분석
4. **기억 잔존도 평가** (`memorability_score`):
   - 라임, 두운, 각운 등 음성적 장치
   - 리듬감 있는 문장 구조
   - 반복 구조 (anaphora, epistrophe)
   - 독특한 조어나 네이밍
   - 짧고 강렬한 태그라인 존재 여부
5. **독창성 평가** (`originality_score`):
   - 해당 카테고리에서 흔히 사용되는 표현과의 차별도
   - 새로운 관점이나 프레이밍
   - `category_benchmarks` 대비 차별화 수준
   - 경쟁 카피(`comparison_copies`)와의 차별도
6. **종합 임팩트 산출**: 5개 항목의 가중 평균을 산출한다.
   - 기본 가중치: attention 0.25, surprise 0.15, emotion 0.25, memorability 0.15, originality 0.20
   - `campaign_objective`에 따라 가중치 조정 가능 (인지도 캠페인이면 attention/memorability 가중)
7. **크리에이티브 팁 생성**: 가장 점수가 낮은 2-3개 항목에 대해 구체적 개선 팁을 제안한다.

## HITL 정책

- **필수 사람 개입 조건**: 없음 (참고 지표이므로 자동 판정에 사용하지 않음)
- **권장 사람 개입 조건**: `overall_impact < 0.4`인 경우 크리에이티브 디렉터에게 카피 재작업 검토 권장
- **승인 역할**: 해당 없음 (참고용 스킬)

## 도구 정책

- **허용 도구**: `docs.read` (카테고리 벤치마크 데이터 조회), `brand_guide.read` (브랜드 크리에이티브 가이드 확인), `policy.read` (크리에이티브 평가 기준 조회), `evidence.read` (과거 캠페인 성과 데이터 조회)
- **승인 필요 도구**: `external.publish` (외부 게시), `bulk.export` (대량 내보내기), `system.write` (시스템 기록)
- **금지 도구**: 경쟁사 카피 직접 크롤링, 비인가 외부 평가 서비스 호출

## 실패 모드

| 실패 상황 | 처리 방법 |
|-----------|----------|
| `copy_text`가 비어 있는 경우 | "평가할 카피를 입력하세요." 반환 |
| `copy_text`에 headline만 있고 body/cta가 없는 경우 | 존재하는 요소만 평가하고 누락 요소 안내 |
| `category_benchmarks`가 없는 경우 | 절대 기준으로 평가 후 "벤치마크 데이터 없이 평가되었습니다" 안내 |
| 극히 짧은 카피 (3자 이하) | "충분한 평가를 위해 더 긴 카피를 입력하세요" 경고와 함께 제한적 평가 |
| 외국어/혼합 언어 카피 | 주 언어 기준으로 평가 후 다국어 요소 안내 |

## 좋은 예시

**입력:**
```json
{
  "copy_text": {
    "headline": "어제의 나와 이별하는 가장 가벼운 방법",
    "body": "무거운 건 선크림이 아니라 어제의 자외선. 오늘부터 에어핏으로 가볍게.",
    "cta": "새로운 나, 시작하기"
  },
  "target_audience": "20-30대 여성, 자기관리 관심",
  "campaign_objective": "브랜드 선호도 강화"
}
```

**출력:**
```json
{
  "attention_score": 0.82,
  "surprise_score": 0.85,
  "emotion_score": 0.88,
  "memorability_score": 0.78,
  "originality_score": 0.83,
  "overall_impact": 0.84,
  "score_rationale": {
    "attention": "이별이라는 감성 키워드로 시선 확보. '가장 가벼운 방법'으로 호기심 유발",
    "surprise": "선크림 광고에서 '이별' 프레이밍은 카테고리 관습을 벗어난 신선한 접근",
    "emotion": "자기 변화/성장 감정 소구. '어제의 나'와 '오늘의 나' 대비로 공감 유도",
    "memorability": "가벼운/무거운 대비 구조가 인상적이나 더 강한 음성적 장치가 있으면 향상 가능",
    "originality": "선크림 = 자기 변화라는 독창적 관점. 기능 나열 대신 감성 스토리텔링"
  },
  "creative_tips": [
    "memorability 향상: headline에 라임이나 두운을 추가하면 기억 잔존도가 높아질 수 있습니다. 예: '가볍게 바꾸고, 가볍게 빛나고'",
    "attention 향상: 숫자나 구체적 데이터를 body에 추가하면 신뢰감과 주목도가 동시에 상승합니다"
  ]
}
```
**이유:** 5개 차원에서 차별화된 점수, 구체적 근거 기반 rationale, 실행 가능한 creative_tips

## 나쁜 예시

**입력:**
```json
{
  "copy_text": {
    "headline": "좋은 선크림 추천",
    "body": "자외선 차단 선크림입니다. 사용해보세요.",
    "cta": "구매하기"
  }
}
```

**출력:**
```json
{
  "attention_score": 0.15,
  "surprise_score": 0.10,
  "emotion_score": 0.12,
  "memorability_score": 0.08,
  "originality_score": 0.10,
  "overall_impact": 0.11,
  "score_rationale": {
    "attention": "일반적이고 평범한 표현으로 스크롤 환경에서 시선을 잡기 어려움",
    "surprise": "카테고리 클리셰 그 자체. 예상 가능한 표현만 사용",
    "emotion": "감정 소구 요소 전무. 정보 전달에 그침",
    "memorability": "기억에 남을 요소 없음. 라임, 리듬, 반복 등 음성적 장치 부재",
    "originality": "모든 선크림 광고에서 볼 수 있는 범용 표현"
  },
  "creative_tips": [
    "attention: '좋은 선크림 추천' 대신 타겟의 구체적 페인포인트를 건드리는 headline을 시도하세요. 예: '자외선은 막고, 피부결은 살리고'",
    "surprise: 선크림을 다른 관점에서 프레이밍해보세요. 기능 나열 대신 사용 경험이나 감정을 중심으로",
    "memorability: 짧고 리듬감 있는 태그라인을 추가하세요"
  ]
}
```
**이유:** 극히 낮은 점수이지만 이는 정보 제공용이며 hard_fail은 발생하지 않음. 구체적 개선 팁을 통해 카피 품질 향상 유도

## 테스트 케이스

| # | 시나리오 | 입력 요약 | 기대 출력 | 판정 |
|---|---------|----------|----------|------|
| 1 | 창의적 카피 | 은유/반전 활용한 고퀄리티 카피 | overall_impact >= 0.7 | PASS |
| 2 | 평범한 카피 | 기능 나열 위주의 일반 카피 | overall_impact < 0.4, creative_tips 3개 이상 | PASS |
| 3 | headline만 입력 | body/cta 없이 headline만 | headline 기준 평가, 누락 요소 안내 | PASS |
| 4 | 인지도 캠페인 | campaign_objective = "인지도 향상" | attention/memorability 가중치 상향 반영 | PASS |
| 5 | 벤치마크 비교 | category_benchmarks 제공 | 벤치마크 대비 상대 평가 반영 | PASS |
| 6 | 다수 카피 비교 | comparison_copies 3개 제공 | 각 카피 대비 상대적 점수 산출 | PASS |
| 7 | 극히 짧은 카피 | "할인" 한 단어 | 제한적 평가 경고 + 가능한 범위 내 점수 | CAUTION |
