## 사용 조건

- 사용자가 "카피 작성", "광고 문구 만들어줘", "카피 생성", "슬로건",
  "배너 문구", "UX 문구", "패키지 문구" 등을 언급할 때 반드시 이 스킬을 사용하세요.
- 브랜드가 지정된 경우 해당 브랜드 보이스 스킬을 함께 참조하세요.
  - LG Electronics: [lg-brand-voice]

## 언어 규칙
- 영어와 한국어 두 가지 버전을 동시에 생성
- 영어는 글로벌 베이스 카피, 한국어는 한국 시장용 카피
- 외래어/영문은 브랜드명, 기술명, 관용적 표현에 한해 허용
- 맞춤법과 띄어쓰기를 정확히 지킬 것

## 인구통계 표현 금지
- 카피 본문에 연령대(예: 30~40대), 소득수준(예: 중산층, 고소득), 사회계층, 직업군을 직접 노출하지 말 것
- 타겟 세그먼트 정보는 톤, 소구점, 창작 전략 결정에만 활용
- 위반 시 hard_fail 판정 대상

## 필수/금지 문구 반영
- 캠페인 브리프에 명시된 필수 포함 문구(mandatory_phrases)는 카피에 반드시 반영
- 금지 문구(forbidden_phrases)는 카피에 절대 사용 금지
- 핵심 메시지(core_message)의 의도를 카피에 자연스럽게 녹여낼 것

## 카피 품질 기준
- 헤드라인: 20자 이내 권장, 한 눈에 핵심이 전달되어야 함
- 본문: 2~3문장, 제품의 핵심 가치를 구체적으로 전달
- CTA: 명확한 행동 유도, 긴급성 또는 혜택 포함 권장
- 근거 없는 수치, 허위 주장, 최상급 표현(업계 최고, 1위 등) 사용 금지

## 창작 원칙
- 각 페르소나의 고유한 창작 스타일을 최대한 살릴 것
- 동일한 브리프에도 페르소나마다 다른 접근이 나와야 함
- 참신하고 기억에 남는 표현을 추구하되, 브랜드 톤에서 벗어나지 말 것

## 생성 프로세스

### Step 1. 브리프 확인

카피를 쓰기 전 아래 항목을 확인합니다. 사용자가 제공하지 않은 항목은 합리적으로 추론하거나 물어봅니다.

| 항목 | 확인 내용 |
|------|----------|
| **카피 유형** | TV/영상, 디지털/SNS, 패키지, UX 중 어느 것인가? |
| **제품/서비스** | 어떤 제품 또는 기능인가? |
| **핵심 메시지** | 전달해야 할 한 가지 핵심은 무엇인가? |
| **타깃** | 누가 읽는가? (연령대, 상황, 감정 상태) |
| **브랜드** | 어떤 브랜드인가? (해당 브랜드 보이스 스킬 참조) |
| **톤 강조점** | 브랜드 보이스 스킬에 정의된 원칙 중 이번 카피에서 특히 강조할 것은? |
| **분량 제약** | 글자 수 제한, 줄 수 제한이 있는가? |

### Step 2. 카피 생성

브리프와 브랜드 보이스 스킬의 원칙을 적용하여 초안을 작성합니다.

### Step 3. 카피 출력 형식

반드시 아래 형식으로 출력할 것 (영어+한국어 동시 생성):

```
[headline_en]
English headline text

[headline_ko]
한국어 헤드라인 텍스트

[body_en]
English body copy text

[body_ko]
한국어 본문 텍스트

[cta_en]
English call-to-action text

[cta_ko]
한국어 행동유도 텍스트

[강조 원칙]
이번 카피에서 중심이 되는 브랜드 원칙
```

- `[headline_en]`, `[headline_ko]`, `[body_en]`, `[body_ko]`, `[cta_en]`, `[cta_ko]`, `[강조 원칙]` 태그를 반드시 포함
- 영어는 글로벌 시장 기준, 한국어는 한국 시장 기준으로 각각 최적화

## 유형별 생성 가이드

### TV/영상 스크립트
- 내레이션은 말하듯 쓴다. 읽히는 글이 아닌 **들리는 글**.
- 오프닝은 질문 또는 일상의 한 장면으로 시작한다

### 디지털/SNS 배너
- 헤드라인 단독으로 의미가 완결되어야 한다
- 모바일 기준 헤드라인 **20자 이내** 권장

### 제품 패키지
- 글자 수 최소화: 핵심 가치 **1~2개**만 담는다

### 홈페이지/앱 UX
- 버튼: 동작(verb) + 기대 가치(benefit) 조합
- 에러 메시지: 사용자 잘못을 지적하지 않고, 해결 방향을 제시

## 자가 검증 체크리스트

카피를 쓴 뒤, 출력 전에 아래를 확인합니다:

- [ ] 헤드라인이 한 눈에 핵심을 전달하는가?
- [ ] 본문이 제품의 핵심 가치를 구체적으로 전달하는가?
- [ ] CTA가 명확한 행동을 유도하는가?
- [ ] 인구통계 정보가 카피 본문에 직접 노출되지 않았는가?
- [ ] 필수 포함 문구가 빠짐없이 반영되었는가?
- [ ] 브랜드 보이스 스킬의 자가 검증 체크리스트를 통과하는가?
