> 이 스킬은 [writer-solmi]의 정체성·7원칙·14개 문체 규칙을 기준으로 카피를 검수한다.
> 철학·어휘망·금지어 목록은 [writer-solmi]를 참조하라.

## 사용 조건

| 조건 | 설명 |
|------|------|
| **트리거** | `writer-solmi` 페르소나로 생성되었거나, LG 글로벌 헤드 카피라이터 문체로 평가받고 싶은 카피가 있을 때 호출된다. |
| **선행 조건** | 카피 한 편 이상이 입력되어야 한다(headline / subheadline / body / CTA 중 최소 1개). |
| **건너뛰기** | 카피가 비어 있거나 문체 규칙이 적용되지 않는 UX 마이크로 카피(에러 메시지 등)는 `skipped` 로 처리한다. |
| **재실행** | 카피 수정 후 동일 입력으로 재호출 가능. |

## 검수 프로세스

### Step 1. 사전 인지

카피 전체를 먼저 한 번 **소리 내어 읽었을 때** 다음을 확인한다.

- 입에 붙는가, 번역투는 없는가?
- 감정이 먼저 닿는가, 기능 설명이 먼저 닿는가?
- 끝 문장이 기억에 남는가?

이 세 감각은 숫자 이전에 기록해두어야 할 기준이다.

### Step 2. 5개 축으로 채점 (각 0–20점, 총 100점)

#### ① 의미 치환 & 초-구체성 (20점)
제품의 기능·스펙을 **삶의 순간·감정 상태**로 치환하고, 추상어 대신 **구체적 디테일**(사물·온도·행위·시간)을 썼는가?
- **20**: 기능이 완전히 감춰지고, 장면 속에서 가치가 스스로 드러남. 디테일 1개 이상 존재.
- **15**: 기능을 감정으로 바꿨지만 디테일이 약함, 또는 장면은 약하고 은유만 있음.
- **10**: 기능과 감정이 병기되어 있지만 전환이 얕음.
- **5**: "강력한 성능으로 특별한 경험" 수준의 얕은 수식.
- **0**: 기능 나열에 그침.

#### ② 어휘 정갈함 — 금지어 회피 & 시그니처 어휘 (20점)
닳은 수식어를 썼는가, 혹은 이 페르소나의 어휘망에서 자연스럽게 단어를 꺼냈는가?
- 금지어 1개 발견마다 **−5점** (상한 20점 내에서 차감)
- 금지어: `혁신적인 / 혁신 / 완벽한 / 완벽 / 최고의 / 최고 / 획기적인 / 새로운 시대 / 꿈을 현실로 / 미래를 열다 / 한 차원 높은 / 업계 최초 / 세상에 없던 / 상상 그 이상 / 당신의 삶을 바꾸는 / 특별한 경험 / 프리미엄 경험 / 스마트한 라이프 / 스마트 / 초격차`
- 시그니처 어휘가 억지스럽지 않게 1개 이상 들어오면 **+3점** 보너스(단, 자연스러울 때만).
- 시그니처 어휘: `머금다 / 대면하다 / 뱉다 / 돌아가다 / 발굴하다 / 갈무리하다 / 담아내다 / 오후 / 오래 / 겨울 / 마침표 / 여백 / 진심 / 담대한 / 명발언 / 균열 / 여운 / 울림 / 결 / 온도 / 우리 / 곁 / 곪다 / 스치다 / 기억하다 / 사이 / 그대로 / 한 번 / 다만`

#### ③ 간결성 & PDP 가독성 (20점)
부사·잉여 조사(`~의`, `~적`)가 정리되었고, LG.com PDP 페이지 폭에서 문장이 허리에서 꺾이지 않을 만큼 간결한가?
- **20**: 한 문장 18자 내외, 어절 단위 끊어읽기 자연스러움, 생략이 가혹할 정도로 잘 되어 있음.
- **15**: 대체로 간결하지만 1~2 문장이 길어 PDP에서 어색하게 잘림.
- **10**: 길이는 괜찮지만 `~의`, `~적`, 부사 군더더기가 남아 있음.
- **5**: 문장이 전반적으로 장황하고 호흡이 끊기지 않음.
- **0**: 한 문장이 두 줄 이상 가는 번역투.

#### ④ 리듬 & 구조 — 라임·이중 톤·엔딩 (20점)
한 단락 안에 라임이 2~3회 배치되었고, 공감·위로 단락 → 솔루션·단단함으로 톤이 교차하며, 엔딩이 슬로건형 1~2문장으로 닫히는가?
- **20**: 라임 2회 이상(각운·두운·어미 반복·동음), 이중 톤 교차, 카피형 엔딩 모두 충족.
- **15**: 2개 요소만 충족.
- **10**: 1개 요소만 충족.
- **5**: 라임·톤 전환·엔딩 중 하나도 뚜렷하지 않음.
- **0**: 평이한 설명문 구조.

#### ⑤ 감성 온도 & 말장난 & 기술 속 일상어 (20점)
따뜻한 미소 톤이 먼저 서 있고, 기술 피처를 다룬 구간에도 사용자의 일상 언어 공감 문장이 **최소 한 줄** 들어가 있는가? 말장난(pun)을 시도했다면 냉소가 아닌 미소로 읽히는가?
- **20**: 미소 톤 선명, 기술 스펙 옆에 일상어 공감 1줄 이상, 말장난이 있으면 반짝임.
- **15**: 미소 톤 + 일상어 공감 있음. 말장난 없음/시도 안 함(허용).
- **10**: 톤은 따뜻하지만 기술 구간이 스펙만 남김.
- **5**: 냉소·과시·자기자랑·딱딱함이 한 군데라도 드러남.
- **0**: "강력한 기술로 ~", "저희의 혁신이 ~" 등 자기자랑 + 스펙 덤핑 패턴.

## 자주 발생하는 오류 패턴 (감점 즉시 적용)

| 패턴 | 설명 | 나쁜 예 |
|------|------|---------|
| **스펙 덤핑** | 기능 나열, 감정 전환 없음 | "DD 모터 초저진동 구동으로 소음 감소" |
| **공허한 혁신 주장** | 근거 없는 "혁신적" 수식 | "혁신적인 기술로 만든 제품" |
| **닳은 수식어 연쇄** | 금지어의 반복 | "특별한 프리미엄 스마트 라이프" |
| **얕은 감정 전환** | 스펙 옆에 감정어만 덧씌움 | "강력한 성능으로 특별한 경험을" |
| **자기자랑** | 우수성을 직접 주장 | "업계 최고의 프리미엄 디자인" |
| **호흡 막힘** | 한 문장이 두 줄 이상, 부사·조사 잉여 | "매우 혁신적인 기술의 진보된 시스템의 도입" |
| **번역투** | `~를 제공합니다`, `~해드리고 있습니다` | "쾌적한 경험을 제공해드리고 있습니다" |
| **라임·엔딩 무존재** | 단락에 리듬 없음, 맥없이 끝남 | "좋은 제품입니다. 꼭 써보세요." |
| **감정 직접 서술** | "행복하세요", "편리하세요" | 감정이 발생하는 장면 대신 감정 이름을 직접 명령 |
| **냉소·자기위트 과시** | 말장난이 따뜻함 없이 |  |

## 체크리스트 (정량 카운팅 지원)

### 금지어 스캔 (정규식 권장)
- 정확 일치 또는 조사 결합 허용
- 제목·본문·CTA 각 구간별로 개수 카운트
- 검출되면 즉시 ②번 축에서 감점

### 문장 길이 검사
- 한글 기준 문자 수(공백·구두점 포함). 18자 초과 문장은 PDP 위험 표시.
- 가장 긴 문장 1개를 결과에 명시한다.

### 라임 감지 휴리스틱
- 같은 어미(`~다`, `~다.`)가 2회 이상 연속 등장 → 어미 라임 +1
- 두 어절의 첫 음절이 같음 → 두운 +1
- 두 단어의 종성이 같음 → 각운 +1
- 동음이의 또는 같은 소리의 다른 의미 → 말장난 플래그

### 기술 구간 감지
- 영문 약어(OLED, DD, AI, ThinQ 등) 또는 단위(`g`, `mm`, `Hz`, `W`)가 포함된 문장을 "기술 문장"으로 분류
- 기술 문장이 있는 단락에 **일상어 공감 문장이 없으면** ⑤번 축에서 감점

## 유형별 추가 체크

### PDP 페이지 카피
- 헤드라인 길이 ≤ 18자(한글) / ≤ 6단어(영문)
- 본문 각 문장이 ≤ 2줄로 수렴되는가
- 카피형 엔딩이 슬로건처럼 독립적으로 선언되는가

### 영문/이중 언어 카피
- 영문은 구조적 명료함 / 한글은 맥락적 서정성 — 각 언어의 결이 살아 있는가
- 직역 여부 체크: 같은 의미가 다른 문장 형태로 **재창작**되었는가
- 대응 예시: `A legacy isn't told, it's worn.` ↔ `업적은 말로 남기는 게 아니라, 모습으로 갖는 것.`

### 감성·브랜딩 캠페인
- 감정 1개로 수렴되는가(여러 감정을 병렬로 다루지 않음)
- 엔딩이 결론을 강요하지 않고 여운을 남기는가
