경쟁 키워드 (competitiveKeywords) 분석
Tavily가 검색하는 것
query_planner 노드에서 LLM이 캠페인 브리프를 기반으로 4개의 영어 검색 쿼리를 자동 생성합니다:

#	검색 초점	예시 (LG OLED TV 캠페인)
1	경쟁사 포지셔닝	"Samsung QD-OLED vs LG OLED TV competitor positioning 2026"
2	소비자 트렌드	"premium TV consumer trends Gen Z millennials 2026"
3	시장 규모/동향	"global OLED TV market size growth forecast 2026"
4	광고/메시지 트렌드	"LG Samsung Sony TV advertising messaging strategy 2026"
Tavily는 각 쿼리당 최대 3개 결과를 가져와 총 최대 12개 웹 페이지의 제목+본문(500자)을 수집합니다.

프롬프트의 필드 가이드라인 (graph.py:341-345):

competitiveKeywords — Language territory already OCCUPIED by competitors. These are words whose meaning is EXHAUSTED.

"word": A keyword/phrase competitors actively own (from web search data)
"count": Relative saturation score 0-100 (100 = completely owned/overused)
Copy should intentionally AVOID or REDEFINE these words.
구체적으로 검색하는 것
수집 대상	내용
경쟁사 광고 카피	Samsung, Sony 등이 사용하는 마케팅 표현
카테고리 공통 언어	"Smart Home", "AI-Powered", "Eco-Friendly" 등 업계 상투적 표현
시장 보고서 키워드	Omdia, IDC 등 리서치 리포트의 반복 용어
소셜/미디어 트렌드	소비자 리뷰, 기사에서 반복되는 키워드
결과 예시

"competitiveKeywords": [
  { "word": "Eco-Friendly", "count": 85 },
  { "word": "Smart Home", "count": 78 },
  { "word": "AI-Powered", "count": 72 },
  { "word": "Sleek Design", "count": 65 },
  { "word": "Energy Saving", "count": 61 }
]
이 키워드들은 **"경쟁사가 이미 선점한 언어 영역"**이므로, 카피라이터는 이 단어들을 피하거나 재정의해야 합니다. count가 높을수록 더 포화된(overused) 표현입니다.

핵심 요약
Tavily 검색 → 경쟁사가 현재 쓰고 있는 광고 언어를 수집 → LLM이 **"이미 의미가 소진된 키워드"**를 추출 → 카피라이터에게 **"피해야 할 단어"**로 제시하는 구조입니다. 직접적으로 "경쟁 키워드"를 검색하는 것이 아니라, 시장 전반의 웹 데이터에서 LLM이 판단하여 추출합니다.
